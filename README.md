# Tidal Power Fitness

A comprehensive fitness management platform for trainers and clients.

## Features
- **User Management**: Profiles, Authentication (JWT)
- **Admin Dashboard**: Manage trainers and classes
- **Workout Tracking**:
  - Create and manage workout templates
  - Real-time active workout tracking
  - Workout history and analytics
- **Scheduling**: Trainer availability and class management

## Tech Stack
- **Frontend**: Next.js, Tailwind CSS
- **Backend**: Node.js, Express, PostgreSQL
- **Deployment**: Render

## Development

### Prerequisites
- Node.js
- PostgreSQL

### Setup
1. Install dependencies:
   ```bash
   cd frontend && npm install
   cd ../backend && npm install
   ```
2. Configure `.env` files in both directories.
3. Run development servers:
   ```bash
   # Frontend
   cd frontend && npm run dev
   
   # Backend
   cd backend && npm run dev
   ```
